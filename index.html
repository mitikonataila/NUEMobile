Coleta e Uso de Dados

A coleta de dados no nosso aplicativo se restringe o uso da função ‘Histórico’, sendo os seguintes dados coletados:

Dados da sua conta (e-mail, senha, faceID ou TouchID)

Dados de uso (frequência de vibração, tempo de uso, frequência cardíaca).

A qualquer momento você pode gerenciar as permissões de coleta de dados do app, baixar ou excluir os seus dados armazenados.
